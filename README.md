# How To Create A Product Card using HTML & CSS | Ecommerce Product Card

Um projeto simples de **card de produto para e-commerce** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **product card moderno**, ideal para lojas online e vitrines de produtos.

## 🛠️ Tecnologias

- HTML
- CSS

## 🎯 Funcionalidades

- Card de produto estilizado
- Layout organizado
- Design moderno e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
